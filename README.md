# 📊 Data/BI Analyst – Tech Assignment (PlatinumRx)

## 📌 Overview

This repository contains solutions for the **Data/BI Analyst Technical Assignment**.
The assignment evaluates proficiency in:

* SQL (data querying & analysis)
* Spreadsheet operations (data transformation & lookup)
* Python (basic problem-solving)

---

## 🗂️ Sections Covered

### 1. 🧠 SQL Proficiency

#### A. Hotel Management System

Worked on relational datasets including:

* `users`
* `bookings`
* `booking_commercials`
* `items`

**Key Problems Solved:**

* Retrieve last booked room per user
* Calculate total billing per booking
* Filter high-value bills for specific months
* Identify most/least ordered items monthly
* Find second highest bill values per month

---

#### B. Clinic Management System

Worked on:

* `clinics`
* `customer`
* `clinic_sales`
* `expenses`

**Key Problems Solved:**

* Revenue by sales channel
* Top 10 valuable customers
* Monthly revenue, expenses, and profit analysis
* Most profitable clinic per city
* Second least profitable clinic per state

---

### 2. 📈 Spreadsheet Proficiency

**Datasets Used:**

* `ticket`
* `feedbacks`

**Tasks Covered:**

* Populate missing columns using lookup logic
* Calculate outlet-wise ticket metrics:

  * Same-day closures
  * Same-hour closures

**Techniques Used:**

* VLOOKUP / INDEX-MATCH / XLOOKUP
* Date-time comparisons
* Conditional aggregation (Pivot Tables)

---

### 3. 🐍 Python Proficiency

**Problems Solved:**

1. **Convert Minutes to Human Readable Format**

   * Example: `130 → 2 hrs 10 minutes`

2. **Remove Duplicate Characters from String**

   * Implemented using loops (without advanced libraries)
